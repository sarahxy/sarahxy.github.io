I was employed as a UI/UX/Web Developer at AXA Rosenberg, a global investment firm. I worked exclusively on the "Rosenberg Visualizer" project, which was launched during my time there.

Rosenberg Visualizer
The Rosenberg Visualizer is a guide to building a well-diversified investment portfolio for clients. You can watch a video explanation here.
The goal of the project was to give confidence to anyone navigating the tool to explore and understand Rosenberg's investment strategy.
Built as a Javascript web application using the HTML5 Canvas element with a NodeJS backend
To bring it to this point, I met with internal stakeholders to learn about the strategy myself, prototyped multiple iterations in Figma, conducted over 40 internal and remote user tests, wrote and revised copy, and implemented my designs in code. We prioritized details that would be intuitive to the user and were consistent with the natural story progression of the tool.
Example of Figma prototyping
I primarily coded in HTML and CSS, and brought the tool up to Chrome Lighthouse's web application standards. Aside from coding new features, I also included Google Analytics tracking, multi-language localization, a responsive mobile-friendly layout, and cross-browser compatibility (including IE).
