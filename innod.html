Innovative Design
Web Development, Web Design
I participated in the Innovative Design club as a member of the Web Innovation Tier for the Spring 2015 semester and a member of the Web Development Tier for the Fall 2016 and Spring 2017 semesters. Over the course of a semester, the tiers take in requests from student groups on campus, and collectively construct websites as a team.
IRC
Our team built a website for the International Relations Council (IRC) at UC Berkeley using Gatsby to help us develop in HTML, SASS, and a React framework. We also implemented Airbnb's CSS / Sass style guide to guide our naming conventions and page structure.
